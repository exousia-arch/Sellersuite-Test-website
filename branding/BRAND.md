# PPCBench — brand guide (v1)

## Logo
- **Primary logomark:** `logomark.svg` — bracketed benchmark bars. Brackets render in
  `currentColor` (set to slate `#0f172a` on light, white `#ffffff` on dark); bars use the
  brand gradient. Pair with the wordmark in a horizontal lockup.
- **App icon / favicon:** `app-icon.svg` — framed bars in a dark rounded square. Use anywhere
  a self-contained, single-tile mark is needed (favicon, PWA, social avatar, OG).
- **Wordmark:** `PPCBench` — `PPC` solid (slate on light / white on dark), `Bench` in the
  brand gradient. Font: Inter, 800 weight, tracking ≈ -0.03em.

### Clear space & sizing
- Keep clear space ≈ the height of one bar around the lockup.
- Minimum logomark size 20px; below that prefer the framed app icon.
- Don't recolor the gradient, stretch, add effects, or place the mark on a busy photo.

## Color
| Token | Hex | Use |
|---|---|---|
| Indigo (brand start) | `#6366f1` | gradient start, primary accent |
| Cyan (brand end) | `#22d3ee` | gradient end, secondary accent |
| Slate 900 | `#0f172a` | dark surfaces / icon frame |
| Slate 50 | `#f8fafc` | light background |
| Emerald `#10b981` · Rose `#f43f5e` · Amber `#f59e0b` | tool accents (Builder / Negator / shared) |

**Brand gradient:** `linear-gradient(90deg, #6366f1, #22d3ee)` (or bottom-left → top-right for the mark).

## Typography
- **Inter** across the board. Headings 800–900 + tight tracking; body 400–500, line-height ~1.6.

## Voice
Precise, trustworthy, fast, a little technical/insider. Lead with the wedge:
*"Amazon PPC tools that never touch your account."*
